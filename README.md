# neroStudy

O **neroStudy** é um projeto pessoal focado em organização de estudos, produtividade e acompanhamento de desempenho.

A ideia do projeto é reunir ferramentas simples para ajudar na rotina de estudos, como controle de tempo, metas, tarefas e registro de progresso.

## Funcionalidades

* Pomodoro para sessões de foco
* Organização de matérias e estudos
* Controle de tarefas
* Registro de acertos e erros
* Acompanhamento de progresso
* Interface simples e objetiva

## Objetivo

Criar um ambiente leve para estudar com mais foco, acompanhar a evolução e manter uma rotina mais organizada.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
